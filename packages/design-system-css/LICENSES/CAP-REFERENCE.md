# Cap widget reference

The `cf-captcha` visual state and motion contract was informed by the public
[Cap widget](https://github.com/tiagozip/cap), copyright © 2025–present Tiago.
Cap is distributed under the
[Apache License 2.0](https://github.com/tiagozip/cap/blob/main/LICENSE).

The cofob component is an independently authored, presentation-only adapter:
it does not include Cap's proof-of-work, instrumentation, networking, worker,
or token logic. Its markup and styling were redesigned around cofob semantic
tokens, accessibility states, and light/dark themes.
